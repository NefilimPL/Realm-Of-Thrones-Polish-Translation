# Realm of Thrones - Polish Translation

## Instalacja (PL)
1. Pobierz to repozytorium lub archiwum z plikami tłumaczenia.
2. Skopiuj folder `ROT-Core` do katalogu `Modules` w miejscu instalacji gry lub moda *Realm of Thrones*.
3. Nadpisz istniejące pliki, jeśli zostaniesz o to poproszony.
4. W launcherze aktywuj moduł **ROT-Core** i ustaw język gry na **Polski**.

## Installation (EN)
1. Download this repository or an archive containing the translation files.
2. Copy the `ROT-Core` folder into the `Modules` directory of your *Realm of Thrones* installation.
3. Overwrite any existing files when prompted.
4. Enable the **ROT-Core** module in the launcher and set the game language to **Polish**.
